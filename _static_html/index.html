Welcome to the MVT Resource Hub!

What is MVT?

Mobile Verification Toolkit (MVT) is a collection of utilities developed by Amnesty International Security Lab to facilitate consensual forensic analysis of Android and iOS devices. Released in July 2021 as part of the Pegasus Project, MVT helps identify traces of compromise by sophisticated mobile spyware attacks.

The toolkit simplifies and automates the process of gathering forensic traces that may indicate potential device compromise. MVT continues to be maintained by Amnesty International and other contributors, serving as an essential tool for technologists, investigators, and security researchers working to protect civil society members at risk of surveillance.

Resources for MVT

Official Documentation and Repositories

  • MVT Official Website: The primary resource for MVT documentation, including installation instructions and usage guides for both Android and iOS devices.
  • MVT GitHub Repository: The official code repository where MVT is maintained, containing the latest updates, issue tracking, and contribution guidelines.
  • MVT on PyPI: The Python Package Index listing for MVT, allowing for easy installation via pip with the command pip3 install mvt.

Guides and Tutorials

  • PiRogue Tool Suite Guide: A detailed guide on using MVT with the PiRogue platform for device forensics, including preliminary setup and example commands.
  • PTS Project Guide: An intermediate guide explaining how to use MVT for analyzing mobile devices and their backup images, emphasizing that MVT requires professional expertise.
  • Privacy Guides - Device Integrity: A resource explaining how MVT can be used to validate device integrity and check for indicators of compromise, with additional suggestions for iOS users at high risk.

Support Services

  • Amnesty International Security Lab Support: Information about Amnesty's digital forensics analysis service for human rights defenders, activists, journalists, and other civil society members at risk. Note that as of April 2, 2025, they are temporarily unable to accept support requests from the general public due to capacity constraints.

Important Notice

MVT is designed for technical users with forensic analysis experience. It is not intended for self-assessment by end users. The tool is licensed to prevent privacy violations and explicitly prohibits extracting or analyzing data from devices without consent.
